# AI Product Card Generator

An AI-powered web application that generates product titles, descriptions, and tags based on a product name and category.  
The app uses Google Gemini AI (`gemini-2.5-flash`) via REST API and displays the result as a styled, responsive product card.

This project is developed as part of **Assignment 2: AI Content Generator for Product Cards**.

## Assignment Overview

### Problem Statement

Create a web application that generates product card details such as title, description, and tags using an AI model and displays the generated content visually in a product card format.

---

## Features

### User Input
- Product Name
- Product Category

### AI-Generated Content
- Product Title
- Short Description
- Relevant Tags

### User Interface
- Styled product card UI
- Loading indicator during AI response
- Fully responsive design for all screen sizes

### Validation and Error Handling
- Input validation for empty fields
- Error handling for API failures

---

## Tech Stack

- **Frontend:** React (Vite)
- **Styling:** CSS
- **AI Model:** Google Gemini 2.5 Flash (Free Tier)
- **API Integration:** REST API (fetch)
- **State Management:** React Hooks (`useState`)

---

## AI Model Used

- **Model Name:** gemini-2.5-flash

### Why this model?
- Free to use with API key
- Fast response time
- Optimized for short text generation
- Suitable for frontend AI-based applications
